Concept Architecture (Developer Game World)

Tujuan step ini adalah membuat fondasi cerita dan struktur dunia portfolio.
Jika ini kuat, design dan coding akan jauh lebih mudah.

Konsep yang kita gunakan:

“The Code Explorer World”

Visitor seperti menjelajah dunia developer kamu.

1. Struktur Dunia Portfolio

Bayangkan halaman website seperti map game kecil.

🌍 Code Explorer World
│
├── 🏠 Base Camp (Hero)
├── ⚒ Skill Forge
├── 🧪 Project Lab
├── ⛰ GitHub Mountain
├── 🧭 Journey Path
└── 🌀 Contact Portal

Setiap section bukan hanya UI, tapi lokasi di dunia developer kamu.

2. Narasi Dunia (Story Layer)

Agar world-building terasa hidup, kita buat narasi sederhana.

Contoh story:

Welcome to my developer world.
A place where ideas are brewed like coffee and turned into code.

Kamu bisa memakai tagline seperti:

Turning Coffee Into Code ☕

Ini cocok dengan background kamu yang punya coffee shop dan roastery.

3. Struktur Section
🏠 Base Camp (Hero)

Ini adalah tempat spawn visitor.

Isi:

avatar developer

headline

short intro

CTA

Contoh:

Headline:

Hi, I'm Firdaus
Full Stack Developer crafting clean and scalable web systems

CTA:

Explore Projects
Enter the Lab

Visual:

developer desk

laptop

coffee mug

floating code symbols

⚒ Skill Forge

Tempat menempa kemampuan developer.

Alih-alih hanya icon tech stack, kita ubah menjadi artifact tools.

Contoh mapping:

Tech	Game Concept
Laravel	Backend Engine
React	UI Builder
Node	Real-time Core
MySQL	Data Vault
Git	Time Machine
🧪 Project Lab

Tempat eksperimen coding.

Setiap project menjadi experiment capsule.

Isi card:

Project Title
Description
Tech Stack
GitHub
Live Demo

Visual style:

laboratory cards

glowing containers

terminal preview

⛰ GitHub Mountain

Visualization commit activity.

Konsep:

GitHub graph = energy grid mountain.

Semakin banyak commit → semakin terang.

🧭 Journey Path

Timeline perjalanan developer.

Misalnya:

2019 — First Lines of Code
2021 — Frontend Engineer
2023 — Full Stack Developer
2026 — Building Scalable Systems

Visual:

adventure path

milestones

🌀 Contact Portal

Tempat kolaborasi.

Isi:

Email
LinkedIn
GitHub
Freelance Inquiry

Visual:

portal / gateway / message station.
