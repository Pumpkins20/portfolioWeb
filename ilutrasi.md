World Illustration System.
Karena kamu memilih scrolling world, pendekatan terbaik adalah setiap section menjadi “scene” di dunia developer kamu. User tetap scroll seperti website normal, tetapi terasa seperti menjelajah map dunia kecil.

Saya juga akan menjaga satu prinsip penting untuk target freelance client:

Scene harus mendukung konten (projects, skills), bukan menggantikannya.

STEP 5 — World Illustration System

Website kamu akan memiliki 6 scene utama.

Struktur world:

Hero → Base Camp

Skill Forge

Project Lab

Journey Path

Contact Portal

Setiap scene memiliki ilustrasi background + mascot pose.
